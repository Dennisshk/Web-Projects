Gästebuchfunktion

Bitte programmiere eine erweiterte Gästebuchfunktion. User sollen in unterschiedliche Text-Feldern auf der Webseite ihren Namen, ihr Alter, ihren Wohnort und eine Nachricht (mehrzeiligen Textfeld) hinterlassen können.

Gestalte die Seite mit CSS.

Wird auf den Absenden-Button geklickt, liest JS alle Felder nacheinander aus, legt die eingegebenen Inhalte auf Variablen ab und generiert damit einen sinnvollen Post, der ebenfalls auf einer Variablen abgelegt werden soll.

Dieser Post soll auf der Webseite mit Zeilenumbruch ausgegeben werden, so dass ein weiterer Eintrag in der Zeile darunter erscheint.

Viel Spaß und Erfolg!
